# EPL Magazine Tracker — Staff Introduction

**Presenter Guide** | March 25, 2026

---

## 1. Logging In

- Open browser → `http://<server>:3000`
- Enter your **email** and **password**
- You'll land on the **Dashboard**

---

## 2. The Sidebar

The left sidebar is your main navigation. It collapses to icons for more screen space.

| Item | Who sees it |
|------|-------------|
| Dashboard | Everyone |
| Magazines | Everyone |
| Manage Magazines | Admin only |
| Transfers | Admin only |
| Reports | Admin only |
| Manage Users | Admin only |
| Audit Log | Admin only |

At the bottom:
- **Branch Selector** — pick which branch you're working at (persists between sessions)
- **Your profile** — name, role badge, and Sign Out button

---

## 3. Dashboard

> "Your daily home screen — what needs attention right now."

- Shows magazines for **your selected branch** only
- Two sections:
  - **Expected This Week** — magazines due to arrive Sun–Sat
  - **Overdue** — magazines past their expected delivery date
- Color-coded summary pills at the top (gold = expected, red = overdue)
- Incoming **transfers** from other branches also appear here

---

## 4. Marking a Magazine as Received

> **Demo: receive a magazine live**

1. Go to **Magazines** in the sidebar
2. Find the magazine in the list
3. Click **Mark Received**
4. Add optional notes (e.g., "damaged cover") → Confirm
5. The dashboard updates — that magazine moves out of "Expected"

---

## 5. Viewing Receipt History

1. From the **Magazines** page, click on a magazine name
2. The detail page shows:
   - Magazine info (cadence, language)
   - Full **receipt history** for your branch — who received it, when, and any notes

---

## 6. Switching Branches

- Use the **Branch Selector** dropdown at the bottom of the sidebar
- The dashboard and magazine lists update immediately to show that branch's data
- Your selection is remembered next time you log in

---

## Admin Features (Admin role only)

### 7. Managing Magazines

> **Manage Magazines** in the sidebar

- **Add** a new magazine — set name, cadence (weekly, monthly, etc.), and language
- **Edit** existing magazines — update any field
- **Deactivate** magazines no longer received (confirmation required)
- **Manage subscriptions** — assign which branches receive which magazines and in what quantity

### 8. Transfers

> **Transfers** in the sidebar

- Create a transfer to move a magazine from one branch to another
- Each transfer goes through a lifecycle: **Pending → Completed** or **Cancelled**
- Filter by status to see what's in progress
- Pending transfers show up on the receiving branch's dashboard

### 9. Reports

> **Reports** in the sidebar — 5 tabs

| Tab | What it shows |
|-----|---------------|
| Receipts | Summary table of all receipts by magazine + bar chart |
| Overdue | Which magazines are late + on-time rate |
| Transfers | Transfer history with status badges |
| Subscriptions | Which branches subscribe to what |
| Timeline | Line chart of receipt frequency over time |

- **Filter** by period (this month, quarter, year, or custom dates), branch, and language
- **Export to Excel** (.xlsx) with the download button

### 10. Managing Users

> **Manage Users** in the sidebar

- Create new staff or admin accounts
- Deactivate users who no longer need access

### 11. Audit Log

> **Audit Log** in the sidebar

- Every action is logged: logins, receipts, magazine changes, user changes
- Searchable and filterable — useful for accountability and troubleshooting

---

## Quick Tips

- **All delete/deactivate actions ask for confirmation** — you won't accidentally remove anything
- **The system calculates "expected" dates automatically** based on the last receipt and the magazine's cadence — no manual scheduling needed
- **Overdue doesn't mean missing** — it just means we haven't logged receipt yet. If the magazine arrived, mark it received!
